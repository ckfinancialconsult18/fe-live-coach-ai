# Call Transcript Archive

## Overview

This file stores annotated transcripts of real sales calls for training and pattern analysis. Each transcript is tagged with the outcome, key moments, techniques used, and coaching notes.

> **Privacy:** All transcripts must be anonymized before archiving. Replace prospect names with fictional names and remove any PII (phone numbers, addresses, policy numbers).

---

## Transcript Template

Use this format when adding a new transcript:

```
## Transcript: [Fictional Name] — [Date] — [Outcome]

**Agent:** [Agent name or initials]
**Duration:** [X min Y sec]
**Outcome:** [Policy Written / Follow-up Scheduled / Not Interested / No Answer]
**Score:** [0–100]
**Lead Type:** [Direct Mail / Inbound / Referral / Aged Lead]
**Carrier Placed:** [Carrier name or N/A]
**Face Amount:** [$X,XXX or N/A]
**Monthly Premium:** [$XX.XX or N/A]

### Key Moments
- [Timestamp or position] — [What happened and why it mattered]

### Techniques Used
- [ ] Need-based discovery
- [ ] Feel/Felt/Found
- [ ] Assumptive close
- [ ] Handling "think about it"
- [ ] Other: [describe]

### Coaching Notes
[What went well, what could be improved, what other agents can learn from this call]

---

[TRANSCRIPT BEGINS]

AGENT: [text]
PROSPECT: [text]
...

[TRANSCRIPT ENDS]
```

## Transcript Tagging System

When searching for a specific scenario, use these tags in the transcript header:

| Tag | Meaning |
|---|---|
| `#objection-price` | Prospect raised a price objection |
| `#objection-spouse` | Prospect needed to consult spouse |
| `#objection-think` | Prospect said "let me think about it" |
| `#objection-not-interested` | Prospect said not interested |
| `#health-diabetic` | Prospect has diabetes |
| `#health-copd` | Prospect has COPD |
| `#health-cancer` | Prospect is a cancer survivor |
| `#replacement` | Call involved replacing existing coverage |
| `#guaranteed-issue` | Prospect placed in GI plan |
| `#graded` | Prospect placed in graded plan |
| `#same-call-close` | Policy written on first call |
| `#callback-close` | Policy written on callback |
| `#lost` | Prospect did not purchase |
| `#no-answer` | No contact made |
