# Carrier Rules & Underwriting Guidelines

## Overview

This file documents the key underwriting criteria, product sweet spots, and watch-out conditions for each contracted carrier. Use it to pre-qualify prospects and select the right carrier before submitting an application.

> **Important:** Underwriting guidelines change frequently. Always verify with the carrier's current underwriting guide before submitting. This file is a field reference, not a substitute for official documentation.

## Guaranteed Issue Carriers

Use guaranteed issue only as a last resort — the 2-year waiting period and higher premiums make it a harder sell. Offer it when:
- Prospect has been declined by all level carriers
- Prospect has active cancer or terminal illness
- Prospect is on dialysis

## Carrier Selection Decision Tree

```
START
│
├── Is the prospect on dialysis or have terminal illness?
│   └── YES → Guaranteed Issue only
│
├── Does the prospect have COPD or use oxygen?
│   └── YES → Corebridge Graded; also quote GI
│
├── Does the prospect have cancer (active or recent)?
│   ├── Active → Guaranteed Issue only
│   └── In remission 5+ years → Foresters
│
├── Is the prospect a tobacco user?
│   └── YES → Transamerica (best tobacco rates)
│
├── Does the prospect have Type 2 Diabetes?
│   └── YES → Americo Eagle (most lenient on diabetes)
│
├── Is the prospect in good health with no major conditions?
│   └── YES → Mutual of Omaha (best rates for healthy seniors)
│
└── Is the prospect female with average health?
    └── YES → Also quote Royal Neighbors (favorable female rates)
```
