## How to Use This Knowledge Base

- **Agents** — Read before live calls to refresh on objection handling, discovery flow, and compliance rules.
- **AI Coach** — These files are chunked and embedded to power real-time coaching suggestions.
- **Trainers** — Update files here; changes propagate to the coaching engine on next sync.

## Update Protocol

1. Make edits directly in the relevant markdown file.
2. Prefix any placeholder line with `<!-- TODO -->` so it can be found with grep.
3. Tag completed sections with `<!-- REVIEWED: YYYY-MM-DD -->`.
